Projeto:

Dashboard de Pedidos: Integração Automatizada ERP Bling API v3 e Power BI

Tecnologias:

Power BI (Dashboards) | Bling ERP | API Bling v3 | Python | DAX

Descrição:

Desenvolvimento de uma solução de dados integrada entre o ERP Bling e o Power BI, com foco na extração, transformação e visualização estratégica dos dados de pedidos.

O projeto iniciou-se com o estudo detalhado da documentação da API v3 do Bling, incluindo os processos de autenticação, segurança e gerenciamento de refresh token para garantir conexões seguras e automatizadas. A coleta dos dados foi realizada diretamente via API, consumindo informações de pedidos em tempo quase real.

Os dados extraídos passaram por um processo de ETL em Python, contemplando tratamento, padronização, modelagem e preparação para análise. Após a carga no Power BI, foram criadas métricas, indicadores e medidas utilizando DAX, possibilitando análises claras sobre vendas, performance operacional e tomada de decisão.

O resultado foi a entrega de dashboards interativos, confiáveis e atualizáveis, proporcionando visibilidade completa dos dados do ERP e apoiando decisões estratégicas baseadas em dados.

Projeto:

Otimização de Dashboards de Operações Logísticas: Integração de Dados Híbridos para Comércio Exterior

Tecnologias:

Oracle Database | SQL | AWS DMS (CDC) | Power BI | DAX | Microsoft Excel

Descrição:

Desenvolvimento de uma solução de Business Intelligence para otimização das operações de Comércio Exterior, centralizando dados transacionais do ERP Oracle e informações complementares mantidas em Excel.

A arquitetura foi desenhada para alta performance, utilizando DMS com CDC para garantir a replicação dos dados em tempo real e a criação de Views diretamente no banco de dados. Essa estratégia moveu a l ógica de transformação para a camada de dados, reduzindo drasticamente a carga de processamento do Power BI.

No front-end, foi estruturado um modelo de dados relacional com métricas avançadas em DAX. O resultado final entregou dashboards automatizados para monitoramento de KPIs operacionais, garantindo maior precisão e a gilidade na tomada de decisão logística.

Projeto:

Dashboard de Expansão e Performance: Gestão de Implantação de Sistemas em Franquias

Tecnologias:

PostgreSQL | Power BI | Power Query | DAX | Microsoft Excel | Google Sheets

Descrição:

Desenvolvimento de uma solução de monitoramento para o processo de expansão de franquias, integrando dados transacionais de implantação armazenados em PostgreSQL com cronogramas e checklists geridos em Excel e Google Sheets.

A solução centraliza o controle de novas unidades, permitindo o acompanhamento do status de cada ativação em tempo real. Foram estruturados indicadores de efetividade dos implantadores, cumprimento de prazos (SLA) e saúde da operação. Através da modelagem no Power BI, o projeto transformou dados dispersos em visões estratégicas, otimizando a alocação de recursos e garantindo a padronização tecnológica nas novas unidades da rede.
